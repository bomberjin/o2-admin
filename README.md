### 项目介绍
- o2-admin是一个轻量级的，前后端分离的Java快速开发平台
- 当前最新版本 
    - server beta 1
    - web beta 1

### 功能介绍
beta 1 版本 预计实现
- 用户管理
    - 机构部门
    - 角色
- 权限管理
    - 部门权限
    - 角色权限
    - 用户权限
    - 权限复用
- 公共参数管理
- 调度管理

### 项目结构
- server 接口服务
- web 前端页面
